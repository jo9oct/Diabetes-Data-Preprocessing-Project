
# Diabetes Data Preprocessing Project

## Overview
This project preprocesses the Pima Indian Diabetes dataset through cleaning, transformation, outlier handling, and class balancing.

## Phases
1. Data Understanding
2. Data Cleaning
3. Transformation
4. Reduction
5. Data Imbalance Handling

## Deliverables
- Final Clean Dataset (`Diabetes-Balanced.csv`)
- Data Dictionary
- Project Report (PDF)

## Technologies
- Python (pandas, numpy, seaborn, matplotlib, scikit-learn, imbalanced-learn)
